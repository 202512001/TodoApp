# TodoApp 🗒️

A full-stack Todo application built with the MERN Stack.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- Add, complete, and delete todos
- User email-based todo management

## How to Run Locally

### Backend
npm install
node server.js

### Frontend
cd client
npm install
npm start
